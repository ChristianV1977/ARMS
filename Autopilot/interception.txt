TargetPosition = InitialPosition + Speed / time
MyPosition = InitialPosition + Speed / time

Tip + Tv / time = Mip + Mv / time
Tip * time + Tv = Mip * time + Mv
Tip * time - Mip * time = Mv - Tv
time * ( Tip - Mip ) = Mv - Tv
time = ( Mv - Tv ) / ( Tip - Mip )
time = relative speed / displacement

Therefore, we can simplify by using relative vectors only.
